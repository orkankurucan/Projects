# ERP System

This is a basic ERP system developed as part of my MSc Software Engineering (Conversion) final project at the University of Westminster.

## Technologies Used
- C#
- Entity Framework
- Azure SQL
- LINQ
- SOLID Principles
- N-layered Architecture
